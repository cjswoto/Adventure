About The Grand Game text adventure engine

The Grand Game is a text adventure engine I am building utilizing ChatGPT to generate the code and images with as little development input by me as possible. I want to see how advanced of a game I can build through this method. I am designing the enviroment so it is all javascript, html and css, with a little php. There is also an Adventure Builder that can be used to build adventures for the engine.

Current Status

The Grand Game engine

The engine is a basic framework at this point, it is driven by a Json file that can be generated by chatgpt or the Adventure Design Tool, which will show you the json structure when you build a scene. The engine supports navigation from one 'room' to another within a 'scene' (aka Adventure) via 'exits' as well as populating the rooms with items and furniture, each of which can have an associated image, description and call to a function to execute code for using an item and interacting(moving) furniture. Items can be picked up into a players inventory and dropped back into a room with the relocation staying peristant within a game session.

The Adventure Designer Tool

The Adventure Designer Tool is just that, a tool you can use to desging your owne adventure. At the moment as a player you can create and upload your own adventure for you and others to play. Images can be linked to for the scenes, items and furniture.

The Tune Tool

Generate melodies for your adventure with the tune tool which uses a simple tone notation to let you but retro style melodies to enhance your adventures. At the moment you can't add them in yet and the tool is still a WIP.
